# News-App-React-Native
News app with react native and nodejs.

#### How To Start API
1. After clone the project open with vsvcode
2. Cd react-app-api
3. npm install
4. npm run dev

#### How To Start React-News-App
1. After clone the project open with vsvcode
2. Cd news-app-react-native
3. npm install
4. npx expo start --tunnel
